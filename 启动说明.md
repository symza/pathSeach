# 路径搜索算法演示应用 - 启动指南

## 启动方法

### 方法一：直接运行（推荐）
在命令行中执行：
```bash
python pathfinding_maze_app.py
```

或者如果系统中有多个Python版本：
```bash
python3 pathfinding_maze_app.py
```

### 方法二：双击运行
直接双击 `pathfinding_maze_app.py` 文件（如果Python已关联.py文件）

## 安装依赖

在启动前，需要先安装必要的依赖包。在项目目录下运行：

```bash
pip install -r requirements.txt
```

或者手动安装：
```bash
pip install matplotlib numpy pillow
```

注意：`networkx` 在 requirements.txt 中列出，但实际代码中可能未使用，可以跳过。

## 系统要求

- Python 3.6 或更高版本
- tkinter（通常随Python一起安装）
- matplotlib
- numpy

## 使用说明

启动应用后，您将看到一个GUI界面，包含：

1. **控制面板**（左侧）：
   - 迷宫参数：大小、复杂度、密度
   - 算法选择：Dijkstra、A*、RRT
   - 控制按钮：生成新迷宫、开始搜索、暂停/继续、重置、显示路径
   - 动画速度控制
   - 信息显示区域

2. **图形显示**（右侧）：
   - 显示迷宫和算法搜索过程的可视化

## 故障排除

如果遇到问题：

1. **缺少模块错误**：运行 `pip install matplotlib numpy pillow`
2. **tkinter 未安装**：
   - Windows：通常已包含
   - Linux：`sudo apt-get install python3-tk`
   - Mac：通常已包含
3. **中文显示问题**：确保系统有中文字体（SimHei、Microsoft YaHei等）
