# 路径搜索算法演示程序

一个基于 Python 和 Tkinter 的图形化路径搜索算法演示程序，支持多种算法可视化。

## 功能特性

- 🎯 **三种路径搜索算法**
  - Dijkstra 算法
  - A* 算法（可调整启发式权重）
  - RRT 算法（可调整步长、目标偏向度等参数）

- 🎨 **可视化界面**
  - 实时显示算法搜索过程
  - 可调整迷宫大小、复杂度、密度
  - 动画速度控制
  - 支持暂停/继续

- 🎬 **GIF 动画生成**
  - 自动生成算法演示动画
  - 播放时间限制在 10 秒内
  - 支持导出为 GIF 文件

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python pathfinding_maze_app.py
```

### 打包为可执行文件

双击运行 `build_exe_utf8.bat`，打包后的 exe 文件在 `dist` 文件夹中。

## 使用说明

1. **生成迷宫**：点击"生成新迷宫"按钮
2. **选择算法**：选择要使用的路径搜索算法
3. **调整参数**：根据需要调整算法参数（A*、RRT）
4. **开始搜索**：点击"开始搜索"查看算法运行过程
5. **生成动图**：点击"生成动图"导出 GIF 动画

## 算法说明

### Dijkstra 算法
经典的最短路径算法，保证找到最短路径。

### A* 算法
启发式搜索算法，通过调整启发式权重可以平衡搜索速度和路径质量。

### RRT 算法
快速随机扩展树算法，适合复杂环境下的路径规划。
